Configure Cards

Watchers

You can define what watchers to show on the cards. For example, it is useful for QA team to use this field to show all the QA engineers so they know every card has a QA assigned.

Watchers names (First name + Last name initial, separated by comma)

Labeling

JIRA labels started with underscore "_" are displayed on the top right corner of the card. e.g. "_InQA", "_FailedQA"

Sub tasks, blocking / blocked tasks

Any sub tasks and blocking/blocked by tasks are displayed on the top left corner of the card.

Hygiene

If the hygiene checkbox is checked, a “Hygiene” label is displayed on the bottom left corner of the card.

Github Integration For Pull Request

If you use Github to track pull requests, enter the Github info below and put the JIRA issue number to the enf of the pull request title.
The plugin would display the pull request label on the bottom right corner of the card.
